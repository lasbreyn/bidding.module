
Product Module  : job_auction
Original Author : Matt Westgate
New Author : Lasbrey Nwachukwu
Settings        : > administer > store > settings > auction

********************************************************************
DESCRIPTION:

Creates a very simple product that can be bid on by users.

You cannot use role-based price adjustments with this module:
( > admin > store > settings > role discounts )
Doing so will mess up your pricing.
